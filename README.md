# 🚗 كاشف أعطال السيارات الفرنسية

## تشغيل المشروع في Termux

### الخطوة 1: تثبيت المتطلبات
```bash
pip install flask
```

### الخطوة 2: الدخول إلى مجلد المشروع
```bash
cd car-diag
```

### الخطوة 3: تشغيل الموقع
```bash
python app.py
```

### الخطوة 4: فتح الموقع في المتصفح
افتح المتصفح واكتب:
```
http://localhost:5000
```

---

## هيكل المشروع
```
car-diag/
├── app.py              ← سيرفر Flask الرئيسي
├── requirements.txt    ← المكتبات المطلوبة
├── data/
│   └── codes.json      ← قاعدة بيانات أكواد الأعطال
└── templates/
    └── index.html      ← واجهة الموقع بالعربية
```

## المرحلة القادمة
- إضافة المزيد من أكواد الأعطال
- نشر الموقع على الإنترنت مجاناً (Render.com)
- ربطه بصفحة فيسبوك
